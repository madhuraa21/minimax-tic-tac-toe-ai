#  **Tic Tac Toe AI using Minimax Algorithm**





####  **Project Description**

###### &nbsp;         This project implements an AI opponent for Tic Tac Toe using the Minimax algorithm.

###### &nbsp;         The AI always makes optimal decisions assuming the human player also plays optimally.

###### 

#### &nbsp;**Features**

###### \- Human vs AI gameplay

###### \- Minimax algorithm based decision making

###### \- Command Line Interface (CLI)

###### \- Win / Loss / Draw tracking

###### 

####  **Tech Stack**
